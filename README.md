# GoIT Python Core - Homework 07

Бот-асистент для управління адресною книгою з підтримкою днів народження.

## Функціональність

### Основні можливості:
- ✅ Додавання нових контактів з телефонами
- ✅ Додавання декількох телефонів до одного контакту
- ✅ Зміна телефонів існуючих контактів
- ✅ Показ телефонів контактів
- ✅ Показ всіх контактів
- ✅ Додавання дати народження
- ✅ Показ дати народження
- ✅ Список найближчих днів народження на тиждень

### Валідація даних:
- Телефонний номер повинен складатися з 10 цифр
- Дата народження у форматі DD.MM.YYYY
- Автоматична обробка помилок з інформативними повідомленнями

## Команди бота

| Команда | Опис | Приклад |
|---------|------|---------|
| `hello` | Вітання від бота | `hello` |
| `add [ім'я] [телефон]` | Додати контакт або телефон | `add John 1234567890` |
| `change [ім'я] [старий] [новий]` | Змінити телефон | `change John 1234567890 0987654321` |
| `phone [ім'я]` | Показати телефони | `phone John` |
| `all` | Показати всі контакти | `all` |
| `add-birthday [ім'я] [дата]` | Додати день народження | `add-birthday John 01.11.1990` |
| `show-birthday [ім'я]` | Показати день народження | `show-birthday John` |
| `birthdays` | Найближчі дні народження | `birthdays` |
| `help` | Показати список всіх команд | `help` |
| `close` або `exit` | Вийти з програми | `exit` |

## Класи

### Field
Базовий клас для полів запису.

### Name(Field)
Клас для зберігання імені контакту (обов'язкове поле).

### Phone(Field)
Клас для зберігання телефону з валідацією (10 цифр).

### Birthday(Field)
Клас для зберігання дати народження з валідацією (DD.MM.YYYY).

### Record
Клас для зберігання інформації про контакт:
- `name`: ім'я контакту
- `phones`: список телефонів
- `birthday`: день народження (опціонально)

Методи:
- `add_phone(phone)`: додати телефон
- `remove_phone(phone)`: видалити телефон
- `edit_phone(old, new)`: змінити телефон
- `find_phone(phone)`: знайти телефон
- `add_birthday(birthday)`: додати день народження

### AddressBook(UserDict)
Клас для управління адресною книгою:
- `add_record(record)`: додати запис
- `find(name)`: знайти запис за ім'ям
- `delete(name)`: видалити запис
- `get_upcoming_birthdays()`: отримати найближчі дні народження

## Особливості реалізації

### Обробка днів народження
- Автоматичний перенос привітань з вихідних на понеділок
- Показ днів народження на наступні 7 днів
- Підтримка високосних років

### Обробка помилок
Декоратор `@input_error` забезпечує:
- Перехоплення всіх помилок
- Інформативні повідомлення для користувача
- Захист від некоректного вводу
